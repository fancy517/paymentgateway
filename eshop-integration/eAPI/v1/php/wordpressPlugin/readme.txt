Plugin installation:

1) copy content of 'www' directory into FTP account of woocommerce plugin
2) place you private key for message signing into directory 'www/monetWebPay/keys'
2) place public key of CSOB payment gateway into directory 'www/monetWebPay/keys'
3) activate and setup module in administration interface of woo commerce (E-shop WooCommerce -> Nastavení -> Pokladna), 
   setup following parameters ...
	- merchant ID (ID obchodníka), setup merchant ID assigned by CSOB
	- payment gateway url (Adresa brány): https://iapi.iplatebnibrana.csob.cz/api/v1
	- payment gateway public key (Veřejný klíč): iplatebnibrana.csob.cz_v20150220.pem
	- merchant private key (Soubor privárního klíče), name of private key file generated by https://iplatebnibrana.csob.cz/keygen tool
	- merchant private key password (Heslo privárního klíče), leave empty (generated private key has no password protection)
